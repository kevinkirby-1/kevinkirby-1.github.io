My Work

East Idaho Forcast is a website designed to show the weather and relevant news for several towns in the south east of Idaho. I created this website to learn and practice HTML, CSS and Javascript. I specifically wanted to gain more experience with json data and APIs. While the weather is updated real time with the OpenWeatherMap API much of the other data is filler json data.

Temple Inn and Suites is a website for a fictitious hotel chain that provides hotels in close proximity to temples for The Church of Jesus Christ of Latter Day Saints. I created this website for to learn and practice HTML, CSS and Javascript. I specifically wanted to practice my website planning and design with this project.

Rock Routs is a web application that allows rock climbers to record and keep track of the routs they have completed or are working on. I created this application as my first introduction to the Django framework, as well as to practice programming in python. My overall goal was to get familiar with how the Django framwork is organized and use it to create something useful.
